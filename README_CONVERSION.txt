## 💻 PLATFORM-SPECIFIC COMMANDS


### macOS (zsh/bash)
```bash

conda activate videomama
python3 demo/app.py
```

### Linux (bash)
```bash
python3 scripts/setup.py
source videomama_env/bin/activate
python3 demo/app.py
```
